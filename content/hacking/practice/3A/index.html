Practice Exercises 3A

View all practice exercises in "Hacking Secret Ciphers with Python".
  1. What are the operators for addition, subtraction, and multiplication?
  2. Which is the operator for division: / or \ ?
  3. Which of these is an integer value, and which is a floating point value?
    42
    3.141592
  4. Which of these lines are not expressions?
    4 x 10 + 2
    3 * 7 + 1
    2 +
    42



Answers:
  1. + - *
  2. /
  3. 42 is an integer value, 3.141592 is a floating point value.
  4. 2 + is not a valid expression because there needs to be a value on the right side of the + operator.
