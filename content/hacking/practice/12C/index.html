We have the following dictionary: cheese = {'foo': 42, 'fizz': 'hello', 'spam': 'foo'}

What do the following expressions evaluate to?

  1. len(cheese)
  2. len(cheese['fizz'])
  3. 'foo' in cheese
  4. 'hello' in cheese



Answers:
  1. 3
  2. 5
  3. True
  4. False
