What value do each of the following expressions evaluate to?

  1. len([2, 4])
  2. len([])
  3. len(['', '', ''])
  4. [4, 5, 6] + [1, 2, 3]
  5. 3 * [1, 2, 3] + [9]
  6. 42 in [1, 2, 3]
  7. 42 in [41, 42, 42, 42]



Answers:
  1. 2
  2. 0
  3. 3
  4. [4, 5, 6, 1, 2, 3]
  5. [1, 2, 3, 1, 2, 3 ,1, 2, 3, 9]
  6. False
  7. True
