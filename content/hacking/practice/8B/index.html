In the following programs, is spam a global or local variable? Or are there both a global and local variables named spam?

  1. spam = 42
  2. def foo():
    spam = 42
  3. spam = 42
def foo():
    spam = 42
  4. spam = 42
def foo():
    print(spam)
  5. spam = 42
def foo():
    global spam
    print(spam)



    6. Answers:
    1. Global
    2. Local
    3. Two variables, one global and one local.
    4. Local
    5. Global
